Stori QA Automation Challenge.

Proyecto realizado con el propósito de cumplir con los requerimientos del proceso de selección.

Prerequisitos:
Intérprete Python 3.5+ debe estar instalado en el sistema
Administrador de paquetes pip debe estar instalado en el sistema

Preparación:

Obtener los drivers pertinentes a los navegadores y colocarlos en /drivers

Chrome:
https://chromedriver.chromium.org/downloads

Firefox:
https://github.com/mozilla/geckodriver/releases

Instalar las dependencias del entorno usando el siguiente comando:

pip install -r requirements.txt

Ejecución

Se usa el comando:

python main.py --browser <navegador>	

<navegador> puede ser chrome ó firefox
